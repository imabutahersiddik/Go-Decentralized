Go Decentralized in One-click

The One-Click Go Decentralized is a tool that allows you to decentralize any web pages in one-click.

Please note that decentralized nodes/networks have a limit for the maximum URI length they can serve. If you need to serve a long URL, you may need to create your own node/network.

For more information on creating your own node/network, you can visit the following GitHub repository:

https://github.com/imabutahersiddik/Decentralized-Web
